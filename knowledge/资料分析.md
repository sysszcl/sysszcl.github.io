### 第三节 （一般增长率）

**增长率定义**： 增长率指的是现期量与基期量的差值（即增长量）与基期量之间的比较，即指在基期量的基础上增长了多大的幅度。

【注】： 增长率又称增速、增幅或者增长幅度、增值率等。

【注】：增长率有正有负，增长率为负时表示下降。

```
增长率（r） = 增长量 / 基期量
```

高频易错考点辨析

【百分数与百分点】

```
百分数表示两个量的比例关系，用除法计算
百分点表示百分数的变化，用加减法计算
考察形式：给出一个百分数和百分点，求另外一个百分数，用加减。
```

【增长率与倍数】

```
增长率指比基数多出的比率；
增长率： A/B

A比B多几倍/增长几倍（%）
增长倍数 = 增长率

倍数指两数的直接比值
多倍数 = (A-B)/B = A/B - 1

两者关系： 倍数 = 增长倍数 + 1 = 增长率 + 1
```

【成数、翻番】

```
成数： 几成相当于十分之几（如：20%就是两成）

翻番： 翻一番为原来的2倍；翻两番为原来的4倍；依次类推，翻n番是原来的2^n倍 
```

【增幅、降幅】变化幅度】

```
增长率（增幅，增长幅度，增速）
可正可负，带符号比
大 30% 20% 10% -40% 小

降幅
必须为负，比较绝对值
大-30%  -20%  -10% 小

变化幅度
可正可负，比较绝对值
大 -40% 30% -20%  10% 小
```

##### 增长率计算

识别： 增长/降落+%、几倍（增长速度、增长幅度） ===>增长率

```
(1) 给一个百分数和百分点：高减低加（理解记忆最好）

(2) 给具体量： r = 增长量 / 基期量   根据选项、截位直除
```

多个年（月）分增长率 > 10%

```
增长率 = （现期 - 基期）/ 基期 > 10%  =======》 现期大于1.1基期
```

##### 增长率比较

识别：增长率（增速、增幅）最快、最慢

```
比较方法： （速算技巧：分数比较）

r = 
(1) 当   现期/基期 >= 2 (倍数明显) 用 现期量/基期量  进行比较
(2) 当   现期/基期 < 2(倍数不明显)，用 增长量/基期量  进行比较
(3) 结论： 给现期量和增长量，用 增长量/现期 进行比较
```

##### 增长量计算

题型识别： 增长 + 具体单位

```
增长量 = 现期量 - 基期量 = 基期量*增长率 = 现期量*r/(1+r)

（1）增长量白分化，|r| = 1/n
（2）r为正： 增长量 = 现期量/(n + 1)
    r为负： 增长量 = -现期量/(n - 1)
```

### 第四节（比重）

关键字：占

题型：

一、现期比重

二、基期比重

三、两期比重

题型识别：

A占B的比重；在B中，A所占的比重

常见的问法展示：

> 第三产业占GDP的比重

> 在总收入中，工资收入所占的比重

比重的核心：部分和整体的属性（单位）相同。

```
比重 = 部分/总体  的三量变化

（1）求比重  比重 = 部分/总体

（2）求总体  总体 = 部分/比重

（3）求部分  部分 = 总体*比重
```

概念引申（比重的特殊表述形式）

``` 
（1）增长贡献率 = 部分增长量 / 整体的增长量

（2）利润率 = 利润/收入（资料分析中）     =利润/成本（数量关系中）
```

##### 比重中的饼图问题

```
构图原则：

12点钟方向,根据材料数据一次顺时针排布

方法：

看各部分的大小、倍数关系（1倍、2倍面积这种）
```

##### 基期比重

题型识别：

问题时间在材料之前，占、比重

```
基期比重 = A(1 + b)/B(1 + a)

速算技巧一：多步除法，子母都截，截位直除

速算技巧二：
1、先判断1+b/1+a与1的关系（> , < , =）===>要是有现期比重A/B更好使(重点在基期比重)

2、根据选项差距截位直除A/B

3、充分利用选项，出答案
```

##### 两期比重比较--------升降判断

题型识别：两个时间 + 比重上升

```
公式： 现期比重 - 基期比重 = A/B - A(1+b)/B(1+a) = A(a-b)/B(1+a)

判断升降方法：
a > b,比重上升
a < b,比重下降
a = b,比重不变

注：
（1）a:分子的增长率，b：分母的增长率
（2）比较a、b时需带正负号比较
```

##### 两期比重--------上升/下降 + 百分点

题型识别：两个时间+比重

```
公式： 现期比重 - 基期比重 = A/B - A(1+b)/B(1+a) = A(a-b)/B(1+a)

解题步骤：
（1）先判方向（a > b,上升;a < b,下降）
（2）再定大小：小于|a - b|,因为A/B一定小于1,1/(1+a)往往小于1，所以一定小于|a-b|（重点在百分点）a为正
A(a-b)/B(1+a) < |a-b|
(3) < |a-b|的有多个，a为负，增长率异常大：根据选项差距截位，代入公式估算即可。
```

速算技巧：截位直除

多个相近的数求平均数：削峰填谷

##### 基期平均数

识别：过去时间+平均

```
1、给现期量、增长量： 基期平均数 = 基期的总数/基期的个数

2、给现期量、增长率： 基期平均数 = A(1+b)/B(1+a)

速算：
（1）先判断1+b/1+a与1的大小关系
（2）再根据选项差距，截位直除A/B(题目可能直接给)（重点）
（3）充分利用选项，出答案
```

##### 两期平均比较------升降

识别：题干中涉及两个时间+平均数问法

```
公式：平均数的增长量 =  现期平均 - 基期平均 = A/B - A(1+b)/B(1+a) = A(a-b)/B(1+a)

判断升降方法：
a > b,平均数上升
a < b,平均数下降
a = b,平均数不变

注：
（1）a:分子的增长率，b：分母的增长率
```

##### 两期平均比较------平均数的增长率

识别：平均/每/单位+增加了%

```
公式： r = (a - b) / (1 + b)

a是分子的增长率 b是分母的增长率

做题逻辑：
（1）根据平均数确定分子、分母（谁除以谁）
（2）代入公式： r = (a - b) / (1 + b)

注：当b约等于0，平均数r=a
```

### 第六节（倍数与比值）

##### 现期倍数

识别： 问题时间与材料一致，A是B的多少倍

```
公式： A/B

速算： 截位直除
```

倍数与增长率

1、A是B的几倍： A/B

2、A比B(增长/多/高)  几倍：r=(A-B)/B = A/B - 1   

增长率=增长倍数

倍数=增长倍数+1 = 增长率+1



识别：2018年比2016年的增长率（隔一年，以率求率）

```
公式： r间隔 = r1+r2 + r1*r2

2018年（r1）  2017(r2)   2016(r间隔)

速算：
（1） r 间隔 = r1、r2的绝对值小于10%，选项相差一个百分点以上，r1*r2可以忽略（10%*10%=1%）
（2）化成分数；化成小数
（3）利用选项看答案
```

```
特征：间隔一年，求倍数

两步走：
（1）先求出r间
（2）间隔倍数=r间+1
```

```
特征：间隔一年，求基期

两步走：
（1）先求出r间
（2）间隔基期 = 现期量/(1+r间)
```

##### 混合增长率

识别：部分1+部分2 = 整体的增长率关系

```
线段法口诀：

距离*量=距离*量

1、混合之前写两边，混合之后写中间。
2、距离和量成反比。
```

识别：

1、房产、地产→房地产

2、进口、出口→进出口

3、邮政、电信→邮电

4、上半年、下半年→全年

```
具体一点的判定：求增长率，但缺少直接数据，则考虑是否为混合

判断口诀：
（1）居中但不中（最小r<总体r<最大r）
（2）偏向基期量大的（那个部分的基期大，总体的增速就离谁近）
（3）偏向搞不定，就线段法精算

注：求混合增长率，做题时无基期量，一般用现期量近似代替基期量(只适用于增长率差异不大)
增长率差距大，必须用基期（如一个20% ，另外一个直接1.5倍（150%））
```

##### 线段法的拓展运用

```
增长率 = 增长量/基期量

平均数 = 总量/人数

重点注意：量是分母
        混合增长率的量的基期量
        混合平均数的量是人数
```



**资料分析中，问人数比例，但是无任何人数的数据。**

方法：用混合平均数线段法
